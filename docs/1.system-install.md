## FORMAT PARTITIONS
```

mkswap /dev/sda2

```

## INSTALL SYSTEM
```
mount /dev/sda1 /mnt
mkdir /mnt/home
mount /dev/sda5 /mnt/home
pacstrap /mnt base base-devel
genfstab /mnt >> /mnt/etc/fstab
cat /mnt/etc/fstab
```
## SYSTEM CONFIGURATION
```
arch-chroot /mnt /bin/bash

```

locale-gen

## GRUB INSTALATION
```	
pacman -S grub os-prober
grub-install /dev/sda
grub-mkconfig -o /boot/grub/grub.cfg
exit
umount /mnt/home
umount /mnt
reboot
```
